# Проєкт: Автоматизація тестування програмного забезпечення

## Опис

Цей проєкт демонструє використання Jest для автоматичного тестування простих математичних функцій.

## Як запустити

1. Встановіть залежності:
npm install

2. Запустіть програму:
npm start

3. Запустіть тести:
npm test
